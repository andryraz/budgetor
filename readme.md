# Posu API

# Lib

- prisma
- swagger-ui-express
- openapitools
- bcryptjs

### Run

1. clone
2. install deps
3. create `prisma/dev.db` file
4. run `npm run migrate:dev`
5. run `npm run dev`

### Database state

run `npm prisma studio`

Here, uou can access the [`api documentation`](http://localhost:8080/api-docs) after running the app
